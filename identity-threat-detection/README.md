# Identity Threat Detection & Sign-In Analytics

This branch contains detection logic for identity-based attacks using Microsoft Graph and sign-in logs. These scripts help identify high-risk authentication patterns and suspicious user behavior.

Identity attacks are increasing in frequency and sophistication. This branch provides tools to detect:

   * MFA fatigue attacks

   * Impossible travel

   * Brute-force attempts

   * Sign-in anomalies

These scripts support SOC teams, IAM analysts, and incident responders.
